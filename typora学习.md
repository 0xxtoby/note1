## 6.链接

### 6.1 图片

#### 语法（本地图片）

```
[标题(url)]
```



#### 语法（网络图片）

```
![bilibili](https://www.baidu.com/img/bd_logo1.png?qua=high)
```

#### *效果*

![bilibili](https://www.baidu.com/img/bd_logo1.png?qua=high)

### 6.2 超链接

#### 语法（行内式链接）

```
[百度][https://www.baidu.com/]
```

#### 效果

[百度][https://www,baidu.com/]

#### 语法（参考式链接）

```
[CSDN网址]:https://www.csdn.net/
```

#### 效果

[CSDN网址]:https://www.csdn.net/

#### 语法（自动链接）

```
<https://www.baidu.com/>
```

#### 效果

<https://www.baidu.com/>
